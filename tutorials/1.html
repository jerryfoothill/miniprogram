层叠样式表，用于设计网页样式

<ul> 定义无序列表，<ol> 定义有序列表，<li> 定义列表项。<dl> 定义定义列表，<dt> 定义术语，<dd> 定义描述。

图像

<img> 用于插入图像。src 属性指定图像的 URL，alt 属性提供图像的替代文本。

表格

姓名 年龄 城市

李四 30 上海

<table> 定义表格，<tr> 定义行，<th> 定义表头单元格，<td> 定义数据单元格。

表单

<form> 定义表单，<input> 定义输入字段，<label> 定义标签，<select> 和 <option> 定义下拉列表，<textarea> 定义多行文本输入。

JavaScript 交互

JavaScript 是一种编程语言，用于创建网页上的交互功能。本页面的 JavaScript 代码定义在 script.js 文件中。

点击事件

JavaScript 可以响应用户的点击事件，并执行相应的操作。

表单验证

JavaScript 可以在提交表单之前验证用户输入，确保数据的有效性。
